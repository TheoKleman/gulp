#Gulp starter

## 1. Project initialisation

First of all, **we highly recommand to use this file tree**

But you also can set your own file configuration in `config/general.js` 

```
project
│   README.md
│   .gitignore
│   ...
│
└───gulp
│   │   
│   └─ *this repository*
│       
│
├───src
│   │
│   ├─── scripts
│   │    ...
│   │
│   └─── styles
│        ...    
│
└─── build
    │   index.html
    │   ...
    │
    │
    ├─── js
    │    ...
    │
    └─── css
         ...      
```

## 2. Run tasks

Simply run `gulp` on your terminal
